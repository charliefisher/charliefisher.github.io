Work Experience

Software Engineering Intern

ReDeTec

ReDeTec is a startup that makes an automatic 3D printer filament recycling machine, the ProtoCycler. I was responsible for adding web connectivity to ProtoCyclers to automatically track usage statistics and update firmware. In order to track usage statistics, I developed a SQL database using Microsoft Azure to organize usage statistics uploaded by individual ProtoCyclers. This database was designed with scalability in mind so that as more units are sold, the database can grow effortlessly. I also configured Google Cloud Storage to host the most recent firmware version from which ProtoCyclers can pull. I accounted for future improvements and added the framework to allow ProtoCycler settings to be updated without updating the firmware version. I was also responsible for developing a feature to detect sudden changes in filament diameter and then gather a video recording using a digital microscope. Finally, I was responsible for fixing bugs in existing C++ and Java codebase related to button debouncing, file I/O, display scaling, and ProtoCycler user interface.

Pointillism Art Plotter MTE 100

Previous Next

In MTE 100, our culminating project was a robot that could automatically plot pointillism art in either black and white or grayscale given any input image. I contributed to both the mechanical design and the software algorithms used to plot an image and improved my skills debugging electromechanical systems.
